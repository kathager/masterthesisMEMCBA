## MEM_masterthesis
#Minimum wages in collective bargaining: Evidence from Switzerland's MEM industry.
This repository includes all files used to produce graphs, tablkes, numbers and analysis for the masterthesis on the introduction of minimum wages in the MEM sector in 2013.
The content is structured similar to the thesis. 
Data on opting out and CBA-membership is in folder 0. 
Data preparation of the main data sources (ESS data enriched with KOF CBA dataset, inflation data, data on spatial units) is folders 1 and 2. 
Descriptive graphs and tables based on main data sources are in folder 5.
The main causla analysis are folders 5. 
The alternative specifications (DID and sub-sectoral analysis) are folder 6 and 7 respectively.
